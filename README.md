# DevBox - 开发者工具箱

免费在线开发者工具箱，所有工具在浏览器本地运行，**不上传任何数据到服务器**。

## 功能

| 工具 | 说明 |
|------|------|
| JSON 格式化 | 格式化、压缩、校验 JSON 数据 |
| 时间戳转换 | Unix 时间戳与日期时间互转 |
| Base64 编解码 | Base64 编码/解码，支持 UTF-8 和 Hex |
| URL 编解码 | URL 编码/解码 |
| 正则测试 | 在线测试正则表达式匹配 |
| 文本差异对比 | 对比两段文本的行级差异 |
| 颜色转换 | HEX、RGB、HSL 颜色格式互转 |

## 技术栈

- Vite
- Vanilla JavaScript
- diff (文本差异对比)

## 开发

```bash
npm install
npm run dev
npm run build
npm run preview
```

## 部署

本项目通过 GitHub Actions 自动部署到 GitHub Pages。

1. Fork 本仓库
2. 在仓库 Settings > Pages 中选择 GitHub Actions 作为源
3. 推送代码到 main 分支，自动触发部署
