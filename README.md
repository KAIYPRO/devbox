## ✨ 特点

- **🔒 隐私安全** — 所有工具在浏览器本地运行，数据不上传任何服务器
- **⚡ 即时响应** — 输入即转换，无需点击按钮等待
- **🌙 深色模式** — 支持深色/浅色主题切换
- **📱 响应式设计** — 桌面端和移动端均可使用
- **📋 一键复制** — 所有输出结果均可一键复制到剪贴板
- **💯 完全免费** — 无限制、无水印、无隐藏收费

## 🚀 本地开发

```bash
# 克隆仓库
git clone https://github.com/KAIYPRO/devbox.git
cd devbox

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```
